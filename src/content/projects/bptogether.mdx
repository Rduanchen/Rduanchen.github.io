# 專案介紹(for end user)

這是一個血壓監測系統的前端應用程式，可以記錄血壓，並將資料分享給親友，也可以設定血壓過高的警告

# Features (功能)

- 現代化的UI
- 記錄血壓
- 分享血壓資料給親友
- 設定血壓過高的警告
- 支援多種語言
- 支援多種裝置


# 技術細節(for developer)
- 前端：Vue 3, Vue Router, Pinia, Tailwind CSS
- 後端：Express, PostgreSQL
- 部署：Railway, Docker
- 登入實作：Firebase Authentication
- 警告實作：使用Firebase Cloud Messaging (FCM) 發送推播通知給使用者
